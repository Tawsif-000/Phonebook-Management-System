Phonebook Management System (C Language)
This is a professional Phonebook Management System built as a part of the Data Structure course in my 3rd Semester. The project demonstrates the practical implementation of a Doubly Linked List to manage and store contact information efficiently.

🚀 Features

Add Contact: Create and add new contacts to the list.

Search: Find contacts by Name or Phone Number.

Update: Modify existing contact details.

Delete: Remove a specific contact or clear the entire list.

File Handling: Save your contacts to a .txt file and load them back automatically.

Reverse Display: View contacts in reverse order using the advantages of a Doubly Linked List.
